## SQL

### 自動生成された SQL クエリ

データベース操作による結果 (`db_result`) に基づき、自動生成された SQLクエリを `show_query()` で確認できます。


```sql
SELECT
  receipt_no,
  receipt_sub_no,
  CAST(STRPTIME(CAST(sales_ymd AS TEXT), '%Y%m%d') AS DATE) AS sales_ymd
FROM receipt
LIMIT 10
```

### 解答例

このクエリをフォーマットすると、次のようになります。

```sql
SELECT
  receipt_no,
  receipt_sub_no,
  CAST(
    STRPTIME(
      CAST(sales_ymd AS TEXT), '%Y%m%d'
    ) AS DATE
  ) AS sales_ymd
FROM 
  receipt
LIMIT 10
```
