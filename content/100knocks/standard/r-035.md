## Rコード (データフレーム操作)

```r
df_receipt %>% 
  filter(!str_detect(customer_id, "^Z")) %>% 
  summarise(sum_amount = sum(amount), .by = customer_id) %>% 
  filter(sum_amount >= mean(sum_amount)) %>% 
  arrange(desc(sum_amount), customer_id) %>%
```

- `filter(!str_detect(customer_id, "^Z"))` により `customer_id` が `"Z"` で始まるデータを除外します。
   
2. `summarise(sum_amount = sum(amount), .by = customer_id)`  
   - `customer_id` ごとに `amount` の合計を求め、新しい列 `sum_amount` を作成する。

3. `filter(sum_amount >= mean(sum_amount))`  
   - 計算した `sum_amount` の平均以上の顧客を抽出する。

## SQLクエリ


```sql
WITH q01 AS (
  SELECT receipt.*
  FROM receipt
  WHERE (NOT((customer_id LIKE 'Z%')))
),
q02 AS (
  SELECT customer_id, SUM(amount) AS sum_amount
  FROM q01
  GROUP BY customer_id
),
q03 AS (
  SELECT q01.*, AVG(sum_amount) OVER () AS col01
  FROM q02 q01
)
SELECT customer_id, sum_amount
FROM q03 q01
WHERE (sum_amount >= col01)
ORDER BY sum_amount DESC, customer_id
LIMIT 10
```


```sql
WITH customer_sales AS (
  SELECT 
    customer_id, 
    SUM(amount) AS sum_amount
  FROM 
    receipt
  WHERE 
    customer_id NOT LIKE 'Z%'
  GROUP BY 
    customer_id
)
SELECT 
  customer_id, 
  sum_amount
FROM 
  customer_sales
WHERE 
  sum_amount >= (
    SELECT AVG(sum_amount) FROM customer_sales
  )
ORDER BY 
  sum_amount DESC
LIMIT 10
```
